## 本书说明

本项目尝试以更易读的图文方式整理《黄帝内经》相关内容，结合 AI 手绘插图，帮助读者更直观地理解经典文本。

本书适合作为：

- 中医经典入门阅读的辅助材料
- 图文形式的学习参考
- 面向大众的健康知识科普读物

## 重要提示

- 本书内容仅供学习、参考与科普使用，不构成医疗建议。
- 本书不能替代医生诊断、处方或治疗方案。
- 如有身体不适，请及时前往正规医疗机构就诊。
